# Simple ToDo List

## Features
- Add tasks 
- Delete tasks
- Mark completed tasks
- Assign priority levels to task
- Sort tasks by date added, alphabetical order and priority (default: date added)
- Light theme: features orange and blue color scheme
- Dark theme: features black color scheme)

## How to Run
- Clone the repository
- Open `index.html` in your browser
  OR
- view live at https://l-y-t-h-r-u-m.github.io/To-Do-List/

## Technologies Used
- HTML
- CSS
- JavaScript
